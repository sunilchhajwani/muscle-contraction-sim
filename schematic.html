Sarcomere Contraction Simulator — Architecture

3D sliding filament theory visualization · Three.js r128 · Single-file HTML

Physiology Engine

  • • Ca²⁺ release/reuptake kinetics (SR → SERCA)
  • • 5-state cross-bridge cycle FSM
  • • 6-phase state machine (Rest→Relax)
  • • Sarcomere shortening: 2.2 → 1.6 µm
  • • Force = f(cross-bridge count)
  • • Rigor mode (ATP-depleted)

3D Scene Components

  • • Z-lines (vertical mesh plates)
  • • Thick filaments / myosin rods (14)
  • • Thin filaments / actin strands (L+R)
  • • Myosin heads (~196 sphere meshes)
  • • Tropomyosin ribbons (shift on Ca²⁺)
  • • Calcium ion particle system
  • • Sarcoplasmic reticulum (torus)

Interactive Controls

  • • Stimulation slider (0–100%) → AP
  • • ATP normal/depleted toggle (rigor)
  • • Simulation speed (0× – 3× slow-mo)
  • • Play / Pause / Reset
  • • Camera orbit (drag) + zoom (scroll)
  • • Labels on/off toggle
  • • Phase-change tip bar (auto-show)

HUD Output

  • • Sarcomere length (µm, live)
  • • [Ca²⁺] concentration (µmol/L)
  • • Force (mN)
  • • Phase indicator (color-coded)
  • • Force trace (canvas 2D, 300-pt)
  • • Cross-bridge / ATP / shortening stats
  • • Tip bar for teaching guidance
